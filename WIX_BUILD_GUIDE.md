## 3. Font Recommendations

Use an editorial pairing with generous line spacing.

| Use | Preferred Font | Wix Alternative | Settings |
| --- | --- | --- | --- |
| Headings | Cormorant Garamond Medium | EB Garamond or Libre Baskerville | Weight 500, tight letter spacing, line height 1.05-1.15 |
| Body | Jost Light | Avenir Light or Helvetica Neue Light | Weight 300, line height 1.6-1.7 |
| Eyebrows / buttons | Jost Medium | Avenir Medium | Uppercase, letter spacing 0.20-0.31em |

Sizing guidance:

| Element | Desktop | Mobile |
| --- | --- | --- |
| Hero headline | 84-94 px | 50-58 px |
| Section heading | 58-64 px | 42-48 px |
| Lead paragraph | 26-30 px | 23-26 px |
| Body copy | 16-18 px | 16 px |
| Eyebrow / buttons | 11-12 px | 11 px |

## 6. Mobile Layout Instructions

- Build mobile first at 390 px width, then refine tablet and desktop.
- Keep all sections stacked in the exact stated order.
- Use `24 px` horizontal page padding on mobile and `64-72 px` on desktop.
- Use `80-104 px` top/bottom spacing per mobile section; increase to `104-128 px` on desktop.
- Hero should fill at least the mobile viewport and place text in the lower-middle area over a strong dark overlay.
- Keep the section navigation as a single horizontally scrollable anchor row beneath the logo and `Register Interest` button. Do not use a hamburger or dropdown menu.
- Stack all button pairs vertically on narrow mobile; use side-by-side buttons from approximately `560 px`.
- Display experience pillars and weekend days as stacked bordered rows on mobile.
- Display facilitator cards in one column on mobile, two columns on tablet and five columns on wide desktop (from approximately `1100 px`).
- Make FAQ accordions full width with large tap areas and restrained plus/minus icons.
- Stack the registration copy above the form on mobile.

## 7. Button And Form Setup

### Buttons

| Button | Destination | Wix Setup |
| --- | --- | --- |
| Register Interest | Anchor `#register` | Link to the Register Interest section anchor |
| View Experience | Anchor `#experience` | Link to the Experience Pillars section anchor |
| Register partnership interest | Anchor `#register` | Link to the same form; preselect Partnership enquiry if Wix automation allows |

Button styling:

- Primary: sand fill `#ECE2D3`, deep brown text `#483126`, no rounded corners, uppercase Jost Medium, subtle hover fade.
- Secondary: transparent fill, 1 px ivory border, ivory text over the hero.
- Keep buttons quiet and spacious; avoid glow, heavy shadow or animated effects.

### Wix Form

1. Add a Wix Form within the Register Interest section.
2. Include required fields `Full name` and `Email address`.
3. Add an `Interest` dropdown with the three values supplied in the copy section.
4. Set submission confirmation copy to: `Thank you. Your interest has been noted.`
5. Connect submissions to Wix Contacts and apply a `Soul Safari 2026 Interest` label.
6. Enable a notification email to the event owner once an official recipient address is confirmed.
7. Do not add payment or ticket checkout until the June 2026 ticket release is ready.

## 10. Step-By-Step Wix Section Recreation

### Global Setup

1. Create a single blank Wix page and name it `Home`.
2. Set the page background to Ivory `#FBF7EF`.
3. Add the typography styles and colour palette specified above.
4. Turn on smooth anchor scrolling if available in the selected Wix editor.
5. Add section anchors named `about`, `experience`, `flow`, `venue`, `facilitators`, `partners`, `faq` and `register`.

### Section 1: Hero

1. Add a full-width, minimum viewport-height strip with Charcoal as the fallback colour.
2. Add the hero photograph or video as the strip background with a dark overlay of approximately 55-70%.
3. Place the Soul Safari logo, eyebrow, headline, subline and two CTA buttons inside a left-aligned content container.
4. Add a simple top navigation containing anchor links only and the `Register Interest` button.
5. Add the four detail items in a thin bordered bar at the base of the hero; stack two-by-two on mobile.

### Section 2: About Soul Safari

1. Add an Ivory strip with centered copy in a narrow container.
2. Add the eyebrow, heading, lead paragraph and supporting paragraph.
3. Beneath the text, add two large images with generous spacing; stack them on mobile.

### Section 3: Experience Pillars

1. Add a Sand strip.
2. Add the eyebrow and heading above the list.
3. Build five repeatable content blocks with number, heading and short copy.
4. Use vertical dividers on desktop and horizontal dividers when stacked on mobile.

### Section 4: Weekend Flow

1. Add an Ivory two-column strip: image on the left, copy on the right.
2. On mobile, place the image first and text second.
3. Add three repeatable rows for Friday, Saturday and Sunday with fine divider lines.

### Section 5: Venue / Wilderness Setting

1. Add a Sand two-column strip using the lodge image and venue copy.
2. Add the three venue facts below the body copy using simple divided rows.
3. Keep this section informational; do not add map embeds or booking widgets.

### Section 6: Facilitators

1. Add an Ivory strip with centered heading.
2. Add a repeating layout of five cards with image, name, descriptor and one-sentence bio.
3. Keep images rectangular and natural; avoid card backgrounds, carousels or pop-up profiles.

### Section 7: Partners

1. Add an Ivory strip with a subtle top divider.
2. Use a text-and-image layout with the table-setting photograph.
3. Place `Register partnership interest` as a quiet text link to the registration anchor.

### Section 8: FAQ

1. Add a Sand strip.
2. Use a Wix accordion element with the five supplied questions and answers.
3. Style headings in the display typeface with simple plus/minus indicators.

### Section 9: Register Interest

1. Add a full-width Charcoal strip with a low-contrast sunset image background and dark overlay.
2. Place the heading and supporting copy beside the Wix Form on desktop and above it on mobile.
3. Style input fields with transparent backgrounds and fine ivory underline borders.
4. Configure submission behaviour according to the Wix Form instructions above.

### Section 10: Footer

1. Add a compact Charcoal footer with a thin top divider.
2. Place the Soul Safari logo, date/location line, presenter line and three small links.
3. Link `Back to top` to the Hero anchor and keep the external brand links opening in new tabs.

## Final Wix Quality Checklist

- One homepage only, with no separate content pages.
- Navigation links scroll to section anchors on the homepage.
- No dropdown, pop-up or carousel navigation.
- Primary CTA is always `Register Interest`.
- Secondary hero CTA is always `View Experience`.
- Layout feels spacious on mobile and desktop.
- Form submissions are connected to Wix Contacts before publishing.
- SEO title and meta description are applied to the Home page.
